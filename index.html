Vulnerability Scanner for Web Services (VSWS) is an automated web service scanner which scans the web services (Rest & Graph-QL) for vulnerabilities. While scanning the Rest API and Graph-QL API test site, one of the first step is to import the resource URLs into the scanner. Another important step is to provide the documentation or log which show the behavior of the web services.
